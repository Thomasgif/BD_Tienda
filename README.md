# BD_Tienda

En este repositorio se desea crear un sistema de ventas para una tienda, el cual permita llevar un control de los productos, clientes, empleados, proveedores, ventas y pagos.

Se proporcionaran los scripts basicos para la creacion de la base de datos y las tablas, asi como los inserts de prueba.

Despues se desea implementar un sistema de interfaz grafica que permita la interaccion con la base de datos, la cual permitira hacer acciones validas para el negocio, como hacer venta, pagar, agregar producto, etc.

Este proyecto se desea implementar con Python y MySQL.

# Estructura del proyecto

- `BD_Tienda/`
  - `Consulta.sql`: Script para la creacion de la base de datos y las tablas.
  - `Inserts.sql`: Script para la insercion de datos de prueba.
  - `README.md`: Este archivo.
  - `requirements.txt`: (Opcional) Listado de dependencias de Python.
  - `src/gui/`: (Opcional) Codigo fuente de la aplicacion.
